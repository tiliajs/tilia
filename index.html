Simple State Management

Tilia is a little library to help manage state in TypeScript and ReScript apps.

Get Started GitHub 

const store = connect({
  vibes: 'immaculate' 🌈
});

function ShowVibes() {
  useTilia()
  return <div>{store.vibes}</div>
}

function elevateVibes() {
  store.vibes = 'legendary' ✨
}
